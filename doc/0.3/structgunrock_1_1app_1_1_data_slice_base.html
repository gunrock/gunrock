Detailed Description

template<typename SizeT, typename VertexId, typename Value>
struct gunrock::app::DataSliceBase< SizeT, VertexId, Value >

Base data slice structure which contains common data structural needed for primitives.

Template Parameters
SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
VertexIdType of signed integer to use as vertex id (e.g., uint32)
ValueType to use as vertex / edge associated values
