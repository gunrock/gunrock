template<typename VertexId, typename SizeT, typename Value>
struct gunrock::app::hits::HITSProblem< VertexId, SizeT, Value >

HITS Problem structure stores device-side vectors for doing HITS Algorithm on the GPU.

Template Parameters
_VertexIdType of signed integer to use as vertex id (e.g., uint32)
_SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
