template<typename VertexId, typename SizeT, typename Value>
struct gunrock::app::cc::CCProblem< VertexId, SizeT, Value >

Connected Component Problem structure stores device-side vectors for doing connected component computing on the GPU.

Template Parameters
_VertexIdType of signed integer to use as vertex id (e.g., uint32)
_SizeTType of unsigned integer to use for array indexing. (e.g., uint32)
_ValueType of float or double to use for computing BC value.
