template<typename KernelPolicy , typename ProblemData , typename PriorityQueue , typename Functor >
gunrock::priority_queue::__launch_bounds__ ( KernelPolicy::THREADS  ,
KernelPolicy::BLOCKS   
)

Mark the queue index in a lookup table, if multiple queue indices map to same vertex ID, only one (the last one written) will be kept.

Compact the input queue into near far pile, remove the duplicate IDs, append the newly generated far pile at the end of current priority queue and output the vertices in the output queue.

Mark whether the vertex ID is valid in near pile/far pile.
