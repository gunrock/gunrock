Detailed Description

template<int LOG_NUM_ELEMENTS>
struct gunrock::util::reduction::WarpReduce< LOG_NUM_ELEMENTS >

Perform NUM_ELEMENTS of warp-synchronous reduction.

Can be used to perform concurrent, independent warp-reductions if storage pointers and their local-thread indexing id's are set up properly.

Requires a 2D "warpscan" structure of smem storage having dimensions [2][NUM_ELEMENTS].
